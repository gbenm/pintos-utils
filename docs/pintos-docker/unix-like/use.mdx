# Uso

## Iniciar el contenedor
Luego de descargar y configurar pintos-docker sólo tiene que ejecutar

```bash
pintos-docker
```

:::tip
Puede ejecutarlo en todos lados :) (si registró la carpeta pintos/utils en su path)
:::

## Detener el contenedor

El contenedor se queda corriendo en background, por lo que necesita dar un docker stop, para hacer esto simplemente necesita correr:

```bash
pintos-docker stop
```
